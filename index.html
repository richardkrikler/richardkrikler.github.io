      Notability-Viewer

      Notability Note Viewer for the Backup PDFs inside of a file system (e.g. with OneDrive integration in Windows).

      Currently there isn't a Notability program from gingerlabs out there for windows. So I've tried to at least create a simple viewer, by using the backup PDFs from Notability. I am using OneDrive as a backup media, so all the files are directly within the filesystem. Of course I could just search for the file I need and open it, but I really like to browse through my notes, without opening and then closing each note separately.

      My software just lists all files and folders of the backup folder. There you can go into the folders and click onto the file you want to see. You can also use the search bar to search through the currently visible elements.

      Keyboard-Shortcuts

      In this script I am programming various Keyboard Shortcuts, to make the every day life with the computer more efficient. For example the duplication of Chrome-Tabs (Ctrl+Alt+D), or switching through all Chrome-Tabs (Right-Shift / Ctrl+Right-Shift), is accelerating the usage.

      Hotkeys.ahk: general Hotkeys (Chrome, IntelliJ, ...)

      ShowMacrosForProgram.ahk: shows all Hotkeys for the current active program

      appSwitcher.ahk: for switching fast between two programs
